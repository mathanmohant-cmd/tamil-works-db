## File Structure Patterns

### Thirumurai Patterns

#### Thevaram (Files 1-7)
```
Line 1: [Author] - தேவாரம் [Volume]. [Volume name]
Line 3: [Number]. [Temple Name] : பண் - [Musical Mode]

#[verse_number]
[verse lines...]

மேல்

#[next_verse_number]
...
```

#### Thiruvasagam (File 8.1)
```
Line 1: மாணிக்கவாசகர் - திருவாசகம் 8. எட்டாம் திருமுறை

@[number] [Sub-work name]

[Continuous verses OR #[number] marked verses]
மேல்
```

#### Thirukkovayar (File 8.2)
```
Line 1: மாணிக்கவாசகர் - திருக்கோவையார் 8. எட்டாம் திருமுறை

* [Main Section] - [Section Tamil]
*[number] [Subsection]
*[number].[subnumber] [Sub-subsection]

#[verse_number]
[verse lines...]
மேல்
```

### Naalayira Divya Prabandham Pattern
```
[Number]. [ஆயிரம் name]
@[Author] - [Work Name]

#[verse_number]
[verse lines...]

மேல் (optional)

@[Next Author] - [Next Work Name]
...
```

### Thiruppugazh Pattern
```
அருணகிரிநாதர் - திருப்புகழ் [Section].[Name]

#[verse_number]
[verse lines...]

மேல்
```

### Thembavani Pattern
```
@[number] [Section name]

#[verse_number]
[verse lines...]

மேல்
```

### Seerapuranam Pattern
```
* [number] [Kandam name]  OR  [Header with காண்டம்]

*[number]. [Padalam name]
$[kandam].[padalam].[verse_group]  (metadata - skip)

#[verse_number]
[verse lines...]

மேல்
```
